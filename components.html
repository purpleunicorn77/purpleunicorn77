Interactive temple (iPad 10 & MacBook Pro)

The homepage shows the entrance of the temple.
Before entering the temple, read "ABOUT" which gives a summary of the design.
Instruction allows the users to understand the process.
The main page allows users to experience each task. Press to see more.

The interactive temple is created by Sublime text with HTML5, CSS, and JavaScript. Each illustration on the web pages was made with Adobe Illustration.

The goal of the interface design is to facilitate the users to understand more about Taiwanese temple culture.
